Have A Nice Day Trader DATE

History is no indicator of future performance. Don't invest what you can't afford to lose.

24 hours of prices are fetched for SYMBOLS coins from CryptoCompare at a rate of BATCH per minute, therefore it takes MINUTES minutes to process the full set of coins. A library of strategies runs over each batch and a position is created if a strategy returns positively. Positions are closed if the return exceeds SELL_THRESHOLD %, falls below CUT % or 24 hours have elapsed since creation. See the code on GitHub or raise an issue.
